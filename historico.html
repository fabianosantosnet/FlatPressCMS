História do FlatPress CMS

Visão Geral do CMS

O FlatPress é um mecanismo de blog extensível multilíngue compatível com o padrão de código aberto que não requer um banco de dados para funcionar. Os dados são armazenados em arquivos de texto (termo em inglês flatfile).

O gerenciador de conteúdo (do termo em inglês Content Management System - CMS) foi concebido para ser simples, por isso implementou apenas um único usuário. É mais adequado para blogs pessoais ou sites de pequenas empresas. Além de implementar widgets, suas funções podem ser extendidas através de plugins.

Suporta categorias e tags (por meio de plugin), também implementa um sistema de comentários, páginas estáticas e normais (posts) e um gerenciador de midia (com suporte a galeria).

Seu editor utiliza a linguagem de marcação simples chamada BBCode dispensando o uso de HTML.

O sistema foi escrito na linguagem PHP e utiliza o Smarty para os templates.

FlatPress não é “WordPress em arquivos simples”. Os plugins do WordPress geralmente não funcionam no FlatPress, muito embora, com algumas linhas de código possam funcionar.
