O FlatPress CMS, com a primeira versão lançada em 16 de Abril de 2006, foi desenvolvido pelo italiano Edoardo Vacchi conhecido como "NoWhereMan" com base no SimplePHPBlog. O projeto havia sido descontinuado em 2018, porém em novembro do mesmo ano foi transferido para o mantenedor alemão Arvid Zimmermann que o continua desenvolvendo. Baixe a versão mais recente do FlatPress v1.2.1 ou a v1.3 (DEV Teste) com o novo Painel de Administração.

Sobre o projeto

O objetivo do projeto é servir como documentação básica para o FlatPress CMS que é um substituto para o Blogger (blogspot) ou Wordpress. Tem também o intuito de divulgar o CMS para os lusófonos, além de prover informações básicas para desenvolvedores e web designers.

Utilizando os conceitos de indie web este gerenciador de conteúdo ajuda na diminuição da concentração de conteúdo em grandes corporações, além das questões de privacidade, permitindo maior liberdade tanto para usuários quanto desenvolvedores, uma vez que podem instalar o CMS em hospedagem própria e customizarem da maneira que acharem melhor.

Por ser menos visado e não requerer banco de dados apresenta mais segurança que as demais aplicações web.

Devido a ter uma estrutura mais simples, torna-se mais rápido e de fácil manutenção.

FlatPress é um mecanismo para criação de blogs ou sites, leve e fácil de configurar. Requer apenas um servidor web com PHP. É mais adequado para blogs pessoais ou sites de pequenas empresas que tenham apenas um usuário único. Oferece uma API de plugin rica e amigável ao desenvolvedor, um sistema de temas e um modelo baseado no mecanismo de template Smarty.

Presente desde 2006, o FlatPress é maduro e bem testado.

Recursos

  • Suporte a categorias
  • Nova UI na versão beta
  • Suporte a Tags via Plugin
  • Disponível em vários idiomas
  • Fácil de configurar e fazer backup
  • Software livre sob a Licença GNU GPLv2
  • Comentários com proteção contra spam
  • Personalização de temas utilizando o Smarty
  • Editor simples com formatação via tags BBCODE
  • Dados são armazenados em arquivos de texto (flat file)
  • Sistema de plugins com recurso de arrastar-soltar (widgets)
