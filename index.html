Bem Vindo(a) !

O objetivo do projeto é servir como base de documentação para o FlatPress CMS que é um substituto para o Blogger (blogspot) ou Wordpress.

FlatPress é um mecanismo de blog leve e fácil de configurar. Puro e simples, apenas PHP. Nenhum banco de dados necessário! Presente desde 2006, o FlatPress é maduro e bem testado.

Recursos

  • Funciona em arquivos, sem banco de dados
  • Fácil de configurar e fazer backup
  • Poderoso sistema de plugins com suporte a widgets
  • Fácil de personalizar com temas, com tecnologia Smarty
  • Função de comentários com proteção contra spam
  • Software livre sob a Licença GNU GPLv2
  • Disponível em vários idiomas
  • Suporte a categorias
  • Editor simples com formatação via tags BBCODE
